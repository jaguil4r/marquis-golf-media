# Marquis Golf Media

The official website for Marquis Golf Media, a father/son golf content creation team.

## Features

- **Modern Design**: Built with Next.js and Tailwind CSS.
- **Interactive**: Animations powered by Framer Motion.
- **Responsive**: Looks great on mobile and desktop.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This site is configured to deploy to **GitHub Pages** automatically.

1. Push this repository to GitHub.
2. Go to **Settings > Pages** in your GitHub repository.
3. Under **Build and deployment**, select **GitHub Actions** as the source.
4. The included workflow (`.github/workflows/deploy.yml`) will handle the rest.
